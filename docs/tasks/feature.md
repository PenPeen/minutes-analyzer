# 議事録口出しBot 追加機能仕様

現在の基本機能（Gemini 2.5 Flash + Slack + Notion連携）を拡張するための将来的な機能追加リストです。優先度と実装複雑度を考慮して整理しています。

## 🚀 Phase 1: 機能強化

#### 📹 Microsoft Teams対応
- **実装内容**
  - Teams会議の文字起こし連携
  - Teams Webhook通知対応
  - Teams Bot統合

- **技術要件**
  - Microsoft Graph API統合
  - Teams Bot Framework実装
  - OAuth認証実装

#### 🎥 Zoom連携
- **実装内容**
  - Zoom Cloud Recording連携
  - 自動文字起こし取得
  - Zoom App Marketplace配信

- **技術要件**
  - Zoom API統合
  - Webhook受信機能
  - マーケットプレイス申請

#### 💬 他Chatツール対応
- **実装内容**
  - Discord通知対応
  - Microsoft Teams Chat連携
  - Mattermost対応

#### 📋 Notion機能強化
- **実装内容**
  - **高度なデータベース連携**: プロジェクト、メンバー、タグの自動関連付け
  - **テンプレート機能**: 会議種別に応じた議事録テンプレート自動適用
  - **ワークフロー自動化**: Notion内でのタスク進捗管理との完全統合
  - **知識ベース構築**: 過去議事録の自動分類・タグ付け機能
